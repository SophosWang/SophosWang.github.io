# Sophos Wang - Market Sentiment Research

Personal website showcasing quantitative finance and sentiment analysis projects.

## Projects

### SPY Sentiment Tracker
A real-time market sentiment analysis tool combining SPY (S&P 500 ETF) price data with social media sentiment indicators.

**Features:**
- Real-time SPY price tracking (Yahoo Finance)
- CNN Fear & Greed Index integration
- Daily automated sentiment reports
- Data visualization with price-emotion correlation analysis

**Tech Stack:**
- Python
- yfinance, pandas, matplotlib
- CNN Fear & Greed API
- GitHub Pages
